# Sales Data Cleaning and EDA using Python

## Project Overview

This project focuses on **Data Cleaning** and **Exploratory Data Analysis (EDA)** using the **Sample Superstore Sales** dataset. The main objective is to clean the raw dataset, improve data quality, and analyze sales data to discover meaningful business insights.

The project was completed using **Python**, **Pandas**, and **Matplotlib** in **Jupyter Notebook**. During the data cleaning process, missing values, duplicate records, and data consistency were checked using Pandas functions. After cleaning the data, Exploratory Data Analysis (EDA) was performed to understand sales performance, profit trends, category-wise performance, regional sales distribution, and the relationship between sales and profit through different visualizations.

This project demonstrates essential Data Analyst skills, including data cleaning, data analysis, data visualization, and business insight generation.
